Steps to Run the Project Locally

### Run Locally
To start the project:
```bash
npm run dev
```
### Installation
1. Install the project dependencies:
   ```bash
   npm install
   ```

### Run the Development Server
1. Start the project:
   ```bash
   npm run dev
   ```
2. Open your browser and navigate to `http://localhost:3000` to view the application.
